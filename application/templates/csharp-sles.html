    Install .NET Core on SLES 12 SP2

    If you already have .NET Core installed on your machine, skip this step. Install .NET Core using the following commands.

    sudo zypper install libunwind
wget http://download.opensuse.org/source/distribution/13.2/repo/oss/suse/src/icu-53.1-2.3.2.src.rpm
curl -sSL -o dotnet.tar.gz https://go.microsoft.com/fwlink/?LinkID=835027
sudo mkdir -p /opt/dotnet && sudo tar zxf dotnet.tar.gz -C /opt/dotnet
sudo ln -s /opt/dotnet/dotnet /usr/local/bin

    You have successfully installed .NET Core on your SLES machine. You now have everything you need to start writing your C# apps with SQL Server!
